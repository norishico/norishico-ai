# 🎯 ROADMAP: スコアリング精度向上によるオッズ上限撤廃

作成日: 2026-04-13
目的: **のりおのビジョン** 「オッズ上限を撤廃し、下限のみでスコアを信じて買い判定する」

## 現状の課題

v6.6 は各クラスで**狭いオッズ帯**を使って赤字を回避している（3勝 8-11倍、F1 15-33倍、G1/G2 7-10+13-20倍）。
これは**スコアが確率較正されていない**ことを隠す絆創膏であり、本質的解決ではない。

**理想像**:
```
P(win | features) = 予測確率（較正済み）
EV = P(win) × odds
buy if EV > 1.05 and odds >= 下限
```
